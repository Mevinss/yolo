# Blind Navigation YOLO

Скрипт `blind_nav.py` выполняет детекцию объектов YOLO, оценивает расстояние до них и озвучивает голосовые подсказки для навигации.

## Что добавлено

- Датасет опасностей `danger_dataset.json`:
	- приоритеты объектов (`priority`),
	- признаки опасности (`danger`),
	- реальные размеры объектов (`height_m`, `width_m`),
	- коэффициент калибровки дистанции (`distance_scale`).
- Отображение всех объектов YOLO:
	- опасные классы подсвечиваются как предупреждения,
	- неопасные классы показываются отдельно с приоритетом.
- Улучшенная оценка расстояния:
	- гибридная модель по высоте и ширине bbox,
	- учет масштаба кадра через `reference_height_px`,
	- автокалибровка фокуса по эталонному объекту.
- Голосовой помощник с приоритетом:
	- говорит объект, сторону, расстояние, приоритет, риск и уверенность,
	- отдельная фраза, если перед человеком обнаружено препятствие.

## Быстрый запуск

```bash
python blind_nav.py --model yolov8m.pt --danger-dataset danger_dataset.json --show-general
```

## Полезные параметры

- `--danger-dataset danger_dataset.json` - файл с приоритетами и размерами.
- `--danger-priority-threshold 0.6` - порог, после которого класс считается опасным.
- `--alert-distance-m 2.0` - рабочая дистанция голосового предупреждения.
- `--max-voice-alerts 2` - лимит голосовых сообщений за цикл.
- `--focal-px 700` - стартовая фокусная длина (пиксели).
- `--reference-height-px 720` - эталонная высота кадра для модели дистанции.
- `--auto-calibrate --calib-label person --calib-distance-m 2.0` - автокалибровка дистанции.

## Настройка датасета опасностей

Редактируйте `danger_dataset.json` и меняйте параметры класса в блоке `classes`.

Пример:

```json
"obstacle": {
	"name": "Препятствие",
	"height_m": 0.55,
	"width_m": 0.55,
	"priority": 0.9,
	"danger": true,
	"distance_scale": 1.0
}
```

Чем выше `priority`, тем выше шанс, что объект будет проговорен первым.
